HTML / CSS

Was sind HTML und CSS?

HTML und CSS sind maßgeblich dafür verantwortlich, wie Internetseiten auf unserem Endgerät dargestellt werden. HTML (Hypertext-Markup-Language) ist eine Auszeichnungssprache, durch die Texte und anderen Daten strukturiert werden. Das CSS (Cascading Style Sheets) formatiert die durch das HTML strukturierte Daten. Das World Wide Web Consortium (W3C) entwickelt diese beiden Sprachen ständig weiter, sodass die aktuellen Version HTML 5.2 und CSS3 sind.

Was lässt sich mit HTML und CSS programmieren?

Mit HTML und CSS lassen sich Internetseiten erstellen, egal ob eine kleine Seite für den eigenen Verein oder ob eine große Seite für beispielsweise einen Onlineshop wie Amazon. Moderne Internetseiten können nicht nur auf dem Computer dargestellt werden, sondern auch auf mobilen Endgeräten. Mit CSS ist es auch möglich, dass die Seite auf dem Smartphone oder Tablet anders dargestellt wird als auf dem Computer. Gerade in der heutigen Zeit ist es sehr wichtig, dass große Seiten vor allem auch auf Smartphones schön dargestellt werden. Mit HTML5 und CSS3 hat aber auch die Welt der Spiele Einzug in den Browser gefunden, denn durch vielfältige Gestaltungsmöglichkeiten ist es inzwischen möglich Spiele ohne weitere Software im Browser darstellen zu lassen.

Hallo Welt!

Der folgende HTML Code zeigt eine einfach Seite mit dem Inhalt "Hallo Welt!". Probier doch mal den Code in eine HTML Datei (Endung .html) zu kopieren und lass sie Dir im Browser anzeigen.

<html lang="de">
<head>
    <meta charset="UTF-8">
    <title>Hallo Welt</title>
</head>
<body>
<p>Hallo Welt!</p>
</body>
</html>

Mit diesem CSS Code können wir unseren Text blau färben.

p {
    color: #0000ff;
}

Damit wir die Änderung auch sehen können, müssen wir den <head> Tag wie folgt ändern

<head>
    <meta charset="UTF-8">
    <title>Hallo Welt</title>
	<style>
		p {
			color: #0000ff;
		}
	</style>
</head>

Womit wird programmiert?

Um mit HTML und CSS zu programmieren sind nicht zwangsläufig spezielle Programme notwendig, da die einfache Struktur es auch möglich macht mit normalen Texteditoren zu arbeiten. Allerdings gibt es auch für diese beiden Sprachen IDEs, die das Programmieren vereinfachen. Hier ist ein kleiner Überblick:

Programm Infos Kosten

Notepad++ ist ein kostenloser Texteditor für Windows, der unter der GNU General Public License für freie Software lizenziert ist. In der aktuellen Version unterstützt das Programm auch beim Arbeiten mit HTML und CSS durch Ergänzungen und das Färben von beispielsweise HTML Tags. Allerdings unterstützt es nicht weitgreifend beim Programmieren.

Das Programm wird komplett kostenfrei unter der GNU GPL zur Verfügung gestellt.


Download

PhpStorm ist eine kostenpflichtige integrierte Entwicklungsumgebung der Firma JetBrains für die Programmiersprache PHP, mit der sich aber auch HTML und CSS einfach programmieren lassen. Ein durchdachtes Design und die clevere Software helfen aktiv beim Programmieren. Der Code wird farbig sinnvoll hinterlegt und Eingaben automatisch ergänzt, sowie Fehler des Syntax angezeigt.

Das Programm lässt sich kostenlos für 30 Tage testen. Sonst kostet es im ersten Jahr 199€, allerdings können Studenten und Lehrer es kostenlos runterladen.
