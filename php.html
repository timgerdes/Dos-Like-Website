PHP

Was ist PHP?

PHP ist eine Skriptsprache, die für die Erstellung dynamischer Interseiten. PHP (Hypertext Preprocessor) wird serverseitig verarbeitet. Der Quellcode wird also nicht an den Browser gesendet, sondern an einen Interpreter auf dem Webserver. Die Sprache zeichnet sich vor allem dadurch aus, dass sie mit Datenbanken und anderen Internetdiensten kommunizieren kann. PHP ist als freie Software unter der PHP-Lizenz lizenziert und liegt aktuell in der Version 7.2.12 vor.

Was lässt sich mit PHP programmieren?

Da PHP ein Präprozessor ist, findet es auch selten Gebrauch ohne andere Sprachen, wie beispielsweise HTML. So ist es mit PHP beispielsweise möglich die Anfragen der Nutzer zu verarbeiten und die richtigen Antworten rauszusuchen. Facebook beispielsweise arbeitete lange Zeit im Hintergrund mit PHP um beispielsweise für jeden Nutzer dynamisch Bilder oder News-Feed Einträge zu laden. Bei gewöhnlichen Anmeldeseiten werden über HTML die Eingaben abgefragt und an PHP weitergeleitet und beispielsweise mit der Datenbank abgeglichen.

Hallo Welt!

Mit dem folgenden Code kannst Du die Ausgabe "Hallo Welt!" erzeugen:

<?php
    echo 'Hallo Welt!';
?>

Wenn Du Dir angucken möchtest, wie eine kleine Software in PHP aussieht, dann guck Dir doch mal eins meiner Projekte hier auf Github an.

Womit wird programmiert?

Da der Quellcode Deiner PHP-Software schnell komplizierter werden kann, empfiehlt es sich eine IDE zu verwenden, aber das Programmieren mit einem simplen Texteditor ist auch möglich.

Programm Infos Kosten

Notepad++ ist ein kostenloser Texteditor für Windows, der unter der GNU General Public License für freie Software lizenziert ist. In der aktuellen Version unterstützt das Programm auch beim Arbeiten mit HTML und CSS durch Ergänzungen und das Färben von beispielsweise HTML Tags. Allerdings unterstützt es nicht weitgreifend beim Programmieren.

Das Programm wird komplett kostenfrei unter der GNU GPL zur Verfügung gestellt.


Download

PhpStorm ist eine kostenpflichtige integrierte Entwicklungsumgebung der Firma JetBrains für die Programmiersprache PHP, mit der sich aber auch HTML und CSS einfach programmieren lassen. Ein durchdachtes Design und die clevere Software helfen aktiv beim Programmieren. Der Code wird farbig sinnvoll hinterlegt und Eingaben automatisch ergänzt, sowie Fehler des Syntax angezeigt.

Das Programm lässt sich kostenlos für 30 Tage testen. Sonst kostet es im ersten Jahr 199€, allerdings können Studenten und Lehrer es kostenlos runterladen.

Atom ist ein quelloffener Texteditor auf Basis von Electron. Standartmäßig unterstützt er das Programmieren von PHP nicht, allerdings lässt es sich schnell durch ein Plugin der Community zu einer PHP IDE erweitern. Das Projekt ist eng mit Github verzahnt und vereinfacht so vor allem die Arbeit an Projekten, die später dort gehostet werden sollen.

Das Programm wird komplett kosten unter der MIT-Lizenz zur Verfügung gestellt.
