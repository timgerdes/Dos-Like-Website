Java

Was ist Java?

Java ist eine plattformunabhängige Programmiersprache, die sowohl auf Rechnern als auch auf Servern ausgeführt werden kann. Der Quellcode muss dafür erst mit einem Compiler in Java-Bytecode umgewandelt werden, der dann nur in einer virtuellen Java-Maschine ausgeführt werden kann, die auf dem Computer zwangsläufig installiert sein muss. Besonders an der Sprache ist die vielseitige Einsatzmöglichkeiten und die große Anzahl an Schnittstellen. Java wird von Sun Microsystems, bzw. Oracle, entwickelt und liegt inzwischen in der Version 11 vor. Seit Jahren ist Java eine der populärsten Programmiersprachen.

Was lässt sich mit Java programmieren?

Ursprünglich wurden sogenannte Java Webapplets häufig im Webbrowser als Ergänzung zu Internetseite eingebunden, aber durch immer moderner werdendes HTML, CSS und auch Javascript hat dieser Bereich weitgehend an Bedeutung verloren. Der Einsatzbereich von Java ist aber dennoch sehr weitreichend, denn Java findet sich sogar in Kaffeemaschinen oder auch HiFi Geräten. Die meistens Apps auf Android Smartphones sind in Java geschrieben und auch populäre Spiele, wie beispielsweise Minecraft, sind in Java geschrieben.

Hallo Welt!

In Java lässt sich ziemlich einfach mit folgendem Code "Hallo Welt!" ausgeben.

public class HalloWelt {

    public static void main(String[] args) {
        System.out.println("Hallo Welt!");
    }

}

Wenn Dich interessiert, wie eine kleine Software in Java aussieht, dann guck Dir doch mal eins meiner Projekte hier auf Github an.

Womit wird programmiert?

Java lässt sich am leichtestens in einer IDE programmieren, aber auch mit dem Texteditor möglich. Auf jeden Fall wird allerdings das Java Development Kit (JDK) benötigt um den Quellcode zu compilen.
