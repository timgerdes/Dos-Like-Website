Javascript

Was ist Javascript?

Javascript ist eine Skriptsprache für dynamische Inhalte auf Internetseiten, die die Möglichkeiten von HTML und CSS erweitert. Durch die vielseitigen Möglichkeiten fällt Javascript immer wieder negativ auf, vor allem durch nervige Werbe-Popups, aber auch teilweise missbräuchlich nutzbaren Code, da Javascript im Browser ausgeführt wird. Daher bieten immer mehr Browser die Funktion an Javascript zu deaktivieren. Inzwischen findet Javascript aber auch vermehrt Anwendung in serverseitigen Ausführungen, wie beispielsweise Node.js.

Was lässt sich mit Javascript programmieren?

Javascript dient nur als Ergänzung für HTML, es kann also unabhängig von HTML keine Internetseiten darstellen. Allerdings kann es beispielsweise dazu genutzt werden Daten zu validieren, wie beispielsweise in einem Kontaktformular auf einer Internetseite. Außerdem kann Javascript dynamisch Daten nachladen ohne die ganze Seite neu laden zu müssen, was es ermöglicht zum Beispiel Suchvorschläge in einer Suche anzeigen zu lassen. Auch für Browsergames ist Javascript geeignet, da darüber die einzelnen Elemente des Spiels angesteuert und verwaltet werden können.

Hallo Welt!

Mit dem folgenden Code öffnet sich ein kleines Popup, in dem "Hallo Welt!" steht.

alert('Hello, world!');

Damit Du den Code nutzen kannst, musst Du ihn noch in ein HTML Dokument wie folgt einbinden.

<html lang="de">
<head>
    <meta charset="UTF-8">
    <title>Hello World</title>
</head>
<body>
<script>alert('Hallo Welt!');</script>
</body>
</html>

Womit wird programmiert?

Javascript lässt sich mit jedem Texteditor programmieren, allerdings ist geht es leichter von der Hand, wenn Du eine IDE benutzt. Vielleicht ist hier ja was für Dich dabei.

Programm Infos Kosten

Notepad++ ist ein kostenloser Texteditor für Windows, der unter der GNU General Public License für freie Software lizenziert ist. In der aktuellen Version unterstützt das Programm auch beim Arbeiten mit HTML und CSS durch Ergänzungen und das Färben von beispielsweise HTML Tags. Allerdings unterstützt es nicht weitgreifend beim Programmieren.

Das Programm wird komplett kostenfrei unter der GNU GPL zur Verfügung gestellt.

Atom ist ein quelloffener Texteditor auf Basis von Electron. Standartmäßig unterstützt er das Programmieren von Javascript, HTML und CSS, lässt sich aber durch Plugins erweitern. Das Projekt ist eng mit Github verzahnt und vereinfacht so vor allem die Arbeit an Projekten, die später dort gehostet werden sollen.

Das Programm wird komplett kostenlos unter der MIT-Lizenz zur Verfügung gestellt.
